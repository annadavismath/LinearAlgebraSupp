\begin{exercise}
Find all solutions (if any) to the system.  

$$\begin{matrix}
      -2x& +&3y&+&3z&=&-9\\
      3x & -&4y&+&z&= &5\\
      -5x&+&7y&+&2z&=&-14
    \end{matrix}$$

 $$(\answer{-15t-21}, \answer{-11t-17}, t)$$
If the system is inconsistent, type DNE in each answer cell.
\end{exercise}
